# ec-auto-scripts

ECサイト業務を効率化するためのPythonスクリプト集です。

---

## 📦 スクリプト一覧

### 🔹 title_rewrite_rakuten.py

楽天の商品タイトルを一括でリライト（自動変換）するためのスクリプトです。

---

## 💡 機能概要

- 楽天RMSから取得した商品データのタイトルを一括編集
- 正規表現・文字置換によるタイトルの最適化
- 編集ルールはコード内に設定済み
- 自動処理後、CSVなどに再出力可能
- Webブラウザを自動操作し、タイトル変更を適用（Selenium）

---

## 🛠 使用技術

- Python 3.x
- Selenium
- BeautifulSoup
- re（正規表現）
- ChromeDriver（パスは自動検出）
